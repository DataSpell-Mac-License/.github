## How to Install

1. Download the latest version of DataSpell for macOS.  
2. Open the DMG installer and move DataSpell.app to the Applications folder.

## Overview

<a href="#">DataSpell for macOS</a> is tailored for data scientists who build analyses and models around notebooks but still want the organisation and power of an IDE. It treats notebooks as first-class citizens, placing them alongside scripts, datasets and environment configuration inside structured projects. Kernel and interpreter management help keep experiments reproducible and separate, which matters when juggling many client projects or research lines. The interface is designed to keep code, output and plots close enough that iteration feels fluid.

Integrated support for Python, major data libraries and visualisation tools lets users go from raw data to insight without constantly switching applications. Notebooks can be converted into scripts or vice versa, depending on the stage of the workflow. With <a href="#">data and variable viewers</a>, it becomes easier to inspect frames, arrays and intermediate results while debugging. Version control integration ensures that results and narrative can be tracked and reviewed just like application code, aligning data work with broader engineering practices.

On macOS, DataSpell aims to coexist with browsers, terminals and communication tools that form the rest of a typical analytics stack. Window layouts can be customised to reserve space for consoles, plots or documentation, depending on preference. Over time, the environment becomes the central place where data professionals on Mac machines read in new data, explore hypotheses and package results into artefacts that can be shared with stakeholders or productionised by engineering teams.

## Major Features
- Notebook-first data science IDE for Python on macOS  
- Project structure that organises notebooks, scripts and data  
- Environment and kernel management for reproducible experiments  
- Integrated support for major data and plotting libraries  
- Variable and data viewers for frames, arrays and objects  
- Tools for turning notebooks into scripts and back again  
- Version control integration for collaborative analytics work  
- Customisable workspace layouts for code, output and plots  
- Compatible with remote and local interpreters alike  
- Helpful for exploratory analysis and modelling pipelines  
- Bridges notebook workflows with engineering practices  
- Designed as a daily driver for Mac-based data scientists
